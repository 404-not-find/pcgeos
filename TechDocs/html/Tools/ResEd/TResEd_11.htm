Resource Editor: 9.6 Resource Editor Menus: Window Menu

| GEOS SDK TechDocs | | 9.5 Utilities Menu

These menu items make manipulation of multiple Translation Files easy, by opening overlapping windows or tiling active windows.

Overlapping :
Choose this item to allow Translation File windows to overlap. This is probably only useful if you are working with more than one such file at a time.
Full-Sized :
Force each Translation File to take up all of Resource Editor's document display area.
Tile :
If you're working with multiple Translation Files, this will lay them out side by side.
