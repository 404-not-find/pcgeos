
List of modules needed for WebMagick 2.0 on the Yago system:

    WEBMAGI2.GEO     WORLD        -- WebMagick 2.0
    WMG3HTTP.GEO     SYSTEM       -- HTTP URL driver
    WMG3FTP.GEO      SYSTEM       -- FTP URL driver
    IJGJPEG.GEO      SYSTEM       -- IJG JPeg convert library
    IMPGRAPH.GEO     SYSTEM\IMPEX -- General import library
    TEX2.GEO         SYSTEM       -- Patched text library

And, of course, all the above modules must be compiled for YAGO.
